## 1. 产品概述

ZenPrompter 是一款专为个人演讲、直播带货、视频录制、公开演示等场景设计的**智能提词器**应用。它提供：

- **流畅的自动滚屏** — 模拟专业提词器体验，可自由调节速度
- **AI 语音跟读** — 通过浏览器语音识别，实时跟随您的朗读语速自动滚动并高亮
- **AI 脚本创作** — 接入大语言模型，根据主题生成口播脚本或润色现有文案
- **丰富的外观定制** — 自定义字体、字号、颜色、背景、透明度等
- **跨平台支持** — 浏览器 Web 版本可直接使用，同时支持打包为 Android / iOS 原生应用

## 3. 核心功能详解

### 3.1 提词显示与滚动

提词区域是应用的核心界面，提供以下能力：

- **自动平滑滚动**：使用 `requestAnimationFrame` 驱动，滚动极其流畅
- **速度调节**：速度范围 0~10，可通过控制面板的 +/- 按钮或速度条精确调节（步进 0.5）
- **播放/暂停**：点击播放按钮或按空格键切换播放状态
- **滚动到底自动停止**：当文本滚动到底部时自动暂停
- **重置**：点击重置按钮立即回到文本顶部
- **阅读模式 vs 编辑模式**：播放时自动进入只读阅读模式，双击进入编辑模式

### 3.2 编辑模式

- **直接编辑**：单击或双击文本区域进入编辑模式，直接在 `contenteditable` 区域修改文本
- **粘贴内容**：支持从剪贴板粘贴纯文本内容
- **拖放导入**：支持拖拽 `.txt` 或 `.md` 文件直接导入文本（详见第 7 节）
- **实时保存**：每次修改自动保存至 localStorage 和 IndexedDB

### 3.3 AI 智能语音跟读

这是 ZenPrompter 最具特色的功能，利用浏览器原生的 **Web Speech API** 实现实时的语音识别与文本追踪。

#### 工作原理

```
用户朗读 → 麦克风采集语音 → Web Speech API 识别 → 文本匹配算法
→ 计算朗读位置 → 高亮当前字词 → 自动滚动保持位置可见
```

#### 启动方式

1. 点击底部控制栏的 **麦克风图标**（位于播放按钮左侧）
2. 首次使用时浏览器会弹出麦克风权限请求，请选择 **"允许"**
3. 启动成功后会显示 **"聆听中..."** 状态提示
4. 开始朗读，提词器会自动跟随您的语速滚动并高亮

#### 跟读视觉反馈

| 元素 | 表现 |
|------|------|
| 已读文本 | 文字变灰色（60% 不透明度），清晰标识朗读进度 |
| 当前朗读位置 | 使用指定高亮颜色（默认黄色）标记 |
| 状态指示 | 底部状态栏显示"聆听中..."/"追踪中..."/"网络错误"等状态 |
| 屏幕中线 | 一条半透明的水平参考线，帮助保持视线位置一致 |

#### 匹配算法

- 使用 **Levenshtein 编辑距离** 进行模糊匹配，容忍最多 35% 的差异
- 优先在当前可见范围内匹配，匹配失败时回退到全文搜索
- 采用滑动窗口机制（窗口大小 40 字符），实时更新位置
- 内置防卡死机制：连续 3 次匹配失败自动尝试重新锁定

#### 注意事项

- **推荐浏览器**：Chrome（对 Web Speech API 支持最好）
- **需要网络**：Web Speech API 需要网络连接以进行语音识别
- **语言支持**：默认识别中文（zh-CN），可修改代码适配其他语言
- **自动重启**：识别因网络波动断开后会自动重连

#### 语音跟读设置

通过控制面板齿轮菜单中的 **"语音跟读设置"** 按钮可详细配置：

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 高亮模式 | 逐字高亮 / 按句高亮 | 逐字高亮 |
| 高亮颜色 | 7 种预设颜色可选 | 黄色 `#FFFF00` |
| 高亮速度 | 高亮动画过渡速度（10~200ms） | 50ms |
| 自动滚动 | 是否跟随朗读位置自动滚动 | 开启 |
| 语音同步 | 语音与高亮是否同步 | 开启 |
| 字体大小 | 语音跟读模式下文本字号（16~48px） | 24px |

### 3.4 AI 创作助手

基于 Gemini 等大语言模型，提供脚本生成和润色功能。

#### 功能入口

点击控制面板右侧的 **✨ AI 创作** 按钮（星星图标）。

#### 生成全新脚本

1. 在输入框中描述您的主题或观点（例如："关于 React 19 新特性的 3 分钟技术分享"）
2. 选择语气风格：
   - **专业学术** — 严谨、逻辑清晰、讲师风格
   - **日常随性** — 自然温和、聊天式轻松风格
   - **趣味幽默** — 幽默风趣、接地气、带网梗
   - **激情促销** — 带货促销、紧迫感口播风格
3. 点击 **"生成全新脚本"** 按钮
4. AI 会自动生成约 250~350 字的口播脚本，立即替换当前提词文本

#### 润色现有文本

1. 确保提词器中已有文本内容
2. 点击 **"润色当前提词器文字"** 按钮
3. AI 会进行以下处理：
   - 将长句拆分为短句（每句 ≤ 20 字）
   - 将书面语替换为口头习惯表达
   - 修复语病和不通顺之处
   - 保留原有段落结构和核心意思

#### 支持的 LLM 引擎

通过 API 设置可选择不同的 AI 模型后端（详见第 5 节），当前支持的引擎包括：

- Google Gemini（默认 / 自定义 Key）
- 硅基流动 SiliconFlow
- DeepSeek 深度求索
- OpenAI GPT-4o
- 月之暗面 Kimi
- 智谱 GLM-4
- 阿里通义千问
- 自定义 OpenAI 兼容 API

### 3.5 脚本库管理

AI 创作助手面板左侧包含一个脚本库（题词库），用于保存和管理多个脚本。

- **保存当前脚本**：点击 **"保存当前"** 按钮将当前提词文本存入库中
- **加载脚本**：点击列表中的脚本标题即可加载
- **修改标题**：悬停脚本条目，点击编辑图标修改标题
- **删除脚本**：悬停脚本条目，点击删除图标移除
- **自动记录**：每条脚本保存时间、字数和标题（取第一行前 30 字）

### 5.1 LLM（大语言模型）配置

| 引擎 | 默认模型 | 特点 |
|------|----------|------|

| 硅基流动 SiliconFlow | DeepSeek-V3 | 国内高性价比算力平台 |

## 7. 文件拖放支持

ZenPrompter 支持从系统文件管理器中直接拖放文件到应用窗口：

## 9. 数据持久化

ZenPrompter 采用双层存储机制确保数据安全：

### 9.1 localStorage（快速读写）

| Key | 存储内容 |
|-----|----------|
| `zen_text` | 当前提词文本内容 |
| `zen_settings` | 所有应用设置（不含播放状态） |
| `tcq_script_library` | 脚本库中的脚本列表 |
| `zen_api_configs` | API 引擎配置信息 |

### 9.2 IndexedDB（结构化存储）

- **数据库名**：`PrompterDB`
- **对象存储**：`prompts`
- **存储结构**：包含文本内容、设置快照、时间戳
- **用途**：为未来多脚本管理和历史记录功能提供基础

### 9.3 自动保存

- 每次文本修改都会自动保存
- 每次设置变更都会自动保存
- 切换模式（编辑/阅读）时也会保存快照

## 12. 常见问题

### Q1: 语音跟读无法启动怎么办？

1. 确认使用的是 Chrome 浏览器
2. 检查是否已授予麦克风权限（地址栏左侧可查看）
3. 确认网络连接正常（右上角有网络状态指示器）
4. 尝试刷新页面后重新启用语音模式

### Q2: AI 生成脚本失败？

1. 检查 API 密钥是否正确配置（打开 API 设置验证）
2. 使用 **"测试连接"** 按钮验证引擎可用性
3. 确认网络连接正常
4. 如果使用默认 Gemini，需确保环境变量 `VITE_API_KEY` 已设置

### Q3: 如何重置所有设置？

清除浏览器 localStorage 即可：
```
打开浏览器开发者工具 → Application → Local Storage
→ 删除 zen_settings、zen_text 等 key
```

### Q4: 文本显示不全或格式错乱？

编辑器为纯文本编辑，不支持富文本格式。建议：
- 粘贴前先清理格式（粘贴为纯文本）
- 使用 Markdown 文件导入时，仅保留文字内容
- 长段落建议手动分段以优化阅读体验

### Q5: 镜像模式下如何操作？

镜像模式只翻转文本显示，控制栏不受影响。开启镜像后仍需面向屏幕操作，提词器玻璃上的文字会自动正向显示。

### Q6: 如何在绿幕/直播场景使用？

1. 将背景透明度设为 0%，背景色设为绿色 (`h=120, s=100%, l=50%`)
2. 调整文字颜色和字号的对比度
3. 使用 OBS 等软件的色度键（Chroma Key）过滤绿色背景
4. 或者直接将背景透明度设为 0，背景设置为纯色

### Q7: 应用支持哪些语言？

- 界面语言：中文（简体）
- 语音识别：中文（普通话），可通过修改代码中的 `recognition.lang` 参数适配其他语言
- AI 生成脚本：中文
