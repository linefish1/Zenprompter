# Design System — Apple Style
## 1. 设计理念
遵循 Apple 极简美学：**简洁、克制、高级、以内容为中心**。
拒绝冗余装饰、拒绝花哨动效、拒绝信息堆砌。

## 2. 色彩规范
### 主色
- White: #FFFFFF
- Black: #000000

### 文字色
- Primary Text: #1D1D1F
- Secondary Text: #86868B

### 品牌色
- Apple Blue: #007AFF
- Hover Blue: #0066CC

### 边框/分割线
- Border: #E5E5E7

## 3. 字体规范
### 中文
- 优先：苹方 / 思源黑体
- 禁用：宋体、黑体加粗、花哨字体

### 英文
- 优先：San Francisco / Inter

### 字号（WEB）
- H1: 48–64px（粗体、居中）
- H2: 24–32px（常规、居中）
- Body: 16px（行高 1.6）
- Caption: 14px（#86868B）

### 字号（Mobile）
- H1: 32–40px
- H2: 18–24px
- Body: 15px
- Caption: 12px

## 4. 间距与圆角
### 间距
- Container Padding: 24px（WEB）/ 16px（Mobile）
- Section Margin: 64px（WEB）/ 48px（Mobile）
- Button Padding: 12px 24px

### 圆角
- 全局圆角：8px（按钮、卡片、输入框）

## 5. 布局原则
### WEB
- 容器宽度：1200px 居中
- 全屏 Banner：100vw，内容居中
- 单页单主题，一屏一重点
- 导航不超过 5 个主菜单

### Mobile
- 宽度：100%，左右 16px 安全边距
- 导航：汉堡菜单
- 内容垂直居中，文字居中对齐

## 6. 按钮样式
### Primary Button
- 背景：#007AFF
- 文字：#FFFFFF
- 圆角：8px
- Hover：#0066CC

### Secondary Button
- 背景：透明
- 文字：#007AFF
- 圆角：8px

## 7. 图片与图标
### 图片
- 背景：纯白 / 纯黑
- 风格：高分辨率、干净光影、产品居中
- 禁止：水印、杂物背景、过度修图

### 图标
- 风格：线性、极简、24px
- 禁止：填充、渐变、立体、卡通

## 8. 交互原则
- 平滑滚动、轻微视差
- 无闪烁、无夸张动效
- 点击反馈柔和
- 无弹窗、无浮窗、无广告

## 9. 文案规范
- 简短、精炼、一句话一个概念
- 中文自然、不直译
- 无感叹号、无营销话术
