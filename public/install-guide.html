安装指南

ZenPrompter 是一款智能提词器应用，支持 Web 浏览器、 Android 和 iOS 三大平台。 本文档将引导您在不同平台上完成安装与配置。

🌐

Web 版

无需安装，打开即用
📱

Android

APK 构建与安装
🍎

iOS

Xcode 构建与部署

环境要求

Web 浏览器版

  • ✓ 现代浏览器（Chrome 90+ / Edge 90+）
  • ✓ 网络连接（用于语音识别与 AI 功能）
  • ✓ 麦克风权限（语音跟读模式需要）
  • i 推荐使用 Chrome 浏览器获得最佳体验

构建开发版

  • ✓ Node.js 18.0+
  • ✓ npm 9.0+
  • ✓ Git
  • ✓ Android Studio（构建 Android 需要）
  • ✓ Xcode 15+（构建 iOS 需要，仅 macOS）

🌐 Web 浏览器版

最简单快捷的使用方式，无需任何安装步骤。

1

打开浏览器

推荐使用 Google Chrome 或 Microsoft Edge 浏览器。

2

访问应用地址

在浏览器地址栏输入 ZenPrompter 的部署地址（如为本地开发，访问 http://localhost:3000）。

3

开始使用

应用加载后，您将看到欢迎指南页面。点击 "开始使用" 按钮进入主界面，即可开始编辑脚本和使用提词功能。

💡 Web 版完全免费，无需注册账号。所有数据存储在您本地浏览器的 localStorage 中，关闭页面后数据不会丢失。

📱 Android 原生构建

将 ZenPrompter 构建为 Android APK 应用，安装到手机或平板上使用。

1

克隆项目代码

git clone https://github.com/linefish1/zn.git
cd zn
2

安装依赖

npm install
3

配置环境变量（可选）

在项目根目录创建 .env 文件：

VITE_API_KEY=你的Google_Gemini_API_Key

如需使用 AI 脚本生成功能，建议配置此 Key。未配置时仍可使用提词器基础功能。

4

构建 Web 资源并同步 Android

npm run build:android

此命令会构建 Web 前端资源，并将它们同步到 Android 原生项目中。

5

在 Android Studio 中打开并构建

npm run open:android

这将启动 Android Studio 并加载 Android 项目。在 Android Studio 中：

  • 连接 Android 设备或启动模拟器
  • 点击 Run 按钮（绿色三角）构建并安装 APK

📌 Android 原生应用需要 麦克风权限，安装时请注意授权。语音跟读功能依赖于设备麦克风。

🍎 iOS 原生构建

将 ZenPrompter 构建为 iOS 应用，在 iPhone 或 iPad 上使用。需在 macOS 环境下进行。

1

克隆项目并安装依赖

git clone https://github.com/linefish1/zn.git
cd zn
npm install
2

构建 Web 资源并同步 iOS

npm run build:ios

构建前端资源并同步到 iOS 原生项目。

3

在 Xcode 中打开

npm run open:ios

在 Xcode 中：

  • 选择您的开发者团队（Signing & Capabilities）
  • 连接 iOS 设备或选择模拟器
  • 点击 Run 按钮构建安装

📌 iOS 构建需要有效的 Apple Developer 账户用于签名。在 Xcode 的 Info.plist 中已包含 NSMicrophoneUsageDescription 麦克风权限声明。

⚡ 开发环境搭建

如果您希望二次开发或本地调试，请按以下步骤搭建开发环境。

1

安装 Node.js

访问 nodejs.org 下载并安装 Node.js 18.0 或更高版本。
验证安装成功：

node --version
npm --version
2

克隆项目并安装依赖

git clone https://github.com/linefish1/zn.git
cd zn
npm install
3

启动开发服务器

npm run dev

开发服务器将在 http://localhost:3000 启动，支持热模块替换（HMR），代码修改后自动刷新。

4

生产构建

构建用于生产部署的静态文件：

npm run build

构建产物输出到 dist/ 目录，可直接部署到任意静态服务器。

5

预览构建结果

npm run preview

在本地预览构建后的生产版本。

环境变量配置

项目使用 .env 文件管理环境变量。

变量名 说明 是否必需
VITE_API_KEY Google Gemini API Key（从 Google AI Studio 获取） 可选

获取 Gemini API Key

  1. 访问 Google AI Studio
  2. 点击 "Get API Key" 按钮
  3. 创建或选择您的 API Key
  4. 将 Key 填入 .env 文件的 VITE_API_KEY 字段

使用自定义 LLM 引擎

ZenPrompter 支持在应用内通过 API 接口设置 面板配置 8 种不同的 LLM 引擎。您无需通过环境变量配置，直接在 UI 界面中填写 API Key 和端点地址即可。

常见安装问题

npm install 失败

  • 确认 Node.js 版本 ≥ 18.0：运行 node --version
  • 尝试清除 npm 缓存：npm cache clean --force
  • 删除 node_modules 和 package-lock.json 后重新安装
  • 国内用户建议使用淘宝镜像：npm config set registry https://registry.npmmirror.com

Android 构建失败

  • 确认已安装 Android Studio 并配置了 Android SDK
  • 在 Android Studio 中打开 SDK Manager，确保已安装 Android SDK 34+
  • 设置 ANDROID_HOME 环境变量指向 SDK 路径
  • 运行 npx cap sync android 重新同步

iOS 构建失败

  • 确认已安装 Xcode 15+ 和 Command Line Tools
  • 在 Xcode 中配置有效的 Apple Developer 签名证书
  • 运行 npx cap sync ios 重新同步
  • 检查 Info.plist 中权限声明是否完整

语音功能不可用

  • 在浏览器中确认已授予麦克风权限（地址栏左侧锁图标查看）
  • Android/iOS 原生应用需在系统设置中开启麦克风权限
  • 语音识别需要网络连接
  • 推荐使用 Chrome 浏览器（Web Speech API 支持最完善）
