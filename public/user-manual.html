ZenPrompter（提词器）是一款专为个人演讲、直播带货、视频录制、公开演示等场景设计的智能提词器应用。它提供流畅的自动滚屏、AI 语音跟读、AI 脚本创作、丰富的外观定制等能力，支持 Web / Android / iOS 三大平台。

React 19 TypeScript Vite 6 Tailwind CSS 4

🚀 快速入门

1

首次打开

打开应用后会看到欢迎界面和简短使用说明，点击 "开始使用" 进入主界面。

2

编辑脚本

双击文本显示区域进入编辑模式，输入或粘贴您的演讲内容。也支持从电脑拖拽 .txt / .md 文件导入。

3

开始提词

点击底部 ▶ 播放按钮或按键盘 空格键 开始自动滚动。使用速度 +/- 按钮调整滚动速度。

4

进阶体验

点击 麦克风图标 启用 AI 语音跟读 — 提词器会实时跟随您的语速自动滚动。点击 星星图标 使用 AI 生成脚本。

提词显示与滚动

提词区域是整个应用的核心界面，采取极简设计，让您专注于脚本内容。

🔄

自动平滑滚动

基于 requestAnimationFrame 驱动的帧率无关滚动引擎，在任何刷新率显示器上均保持匀速平滑。

🎛️

速度调节（0~10）

通过控制面板的 +/- 按钮或速度滑块精确调节，步进 0.5。
速度 2 = 40px/s，速度 5 = 100px/s

⏸️

播放/暂停

点击播放按钮或按 空格键 切换。滚动到底时自动暂停。

⏹️

自动停止

当文本滚动至底部时自动暂停播放，无需手动操作。

↺

重置到顶部

点击左侧重置按钮，立即回到文本开头，方便重新排练。

📖

阅读/编辑双模式

播放时自动进入只读阅读模式（不可误触编辑），双击切换回编辑模式。

编辑模式

●
直接编辑

双击文本区域进入编辑模式，直接在 contenteditable 区域修改文本内容。

●
粘贴内容

支持从剪贴板直接粘贴纯文本内容，自动过滤富文本格式。

●
拖放导入

支持拖拽 .txt 或 .md 文件直接导入文本内容。

●
实时保存

每次修改自动保存至浏览器 localStorage 和 IndexedDB，关闭页面不会丢失数据。

启动方式

  1. 点击底部控制栏的 麦克风图标（位于播放按钮左侧）
  2. 首次使用时浏览器会弹出麦克风权限请求，请选择 "允许"
  3. 启动后会显示 "聆听中..." 状态提示
  4. 开始朗读，提词器自动跟随您的语速滚动并高亮

视觉反馈

元素表现
已读文本文字变灰色（60% 不透明度），清晰标识朗读进度
当前朗读位置使用指定高亮颜色（默认黄色）标记当前字词
状态指示显示"聆听中..."/"追踪中..."/"网络错误"等实时状态
屏幕中线半透明水平参考线，帮助保持视线位置一致

匹配算法特性

  • 使用 Levenshtein 编辑距离 进行模糊匹配，容忍最多 35% 差异
  • 优先在当前可见范围内匹配，失败时回退到全文搜索（双层策略）
  • 采用 40 字符滑动窗口，实时更新朗读位置
  • 内置防卡死机制：连续 3 次匹配失败自动尝试重新锁定
  • 网络错误后 1 秒自动重启语音识别

💡 推荐使用 Chrome 浏览器 体验语音跟读功能。需要网络连接。默认识别中文（zh-CN）。

✨ AI 创作助手

基于大语言模型，一键生成口播脚本或润色现有文案。

功能入口

点击控制面板右侧的 ✨ AI 创作 按钮（星星图标）打开 AI 创作面板。

生成全新脚本

  1. 描述您的主题或观点
  2. 选择语气风格：专业学术 / 日常随性 / 趣味幽默 / 激情促销
  3. 点击 "生成全新脚本"
  4. AI 自动生成 250~350 字的口播脚本

润色现有文本

  1. 确保提词器中已有文本内容
  2. 点击 "润色当前提词器文字"
  3. AI 自动：长句拆短（≤20字）、书面→口语、修复语病
  4. 保留原段落结构和核心意思

数据持久化

ZenPrompter 采用双层存储机制确保数据安全。

localStorage（快速读写）

Key存储内容
zen_text当前提词文本内容
zen_settings所有应用设置
tcq_script_library脚本库列表
zen_api_configsAPI 引擎配置

IndexedDB（结构化存储）

  • 数据库名：PrompterDB
  • 对象存储：prompts
  • 存储结构：文本 + 设置快照 + 时间戳
  • 自动保存：每次文本/设置变更、模式切换均保存

常见问题

Q1: 语音跟读无法启动怎么办？ ▼
  • 确认使用的是 Chrome 浏览器
  • 检查是否已授予麦克风权限（地址栏左侧可查看）
  • 确认网络连接正常（左上角有网络状态指示器）
  • 尝试刷新页面后重新启用语音模式
Q2: AI 生成脚本失败？ ▼
  • 检查 API 密钥是否正确配置（打开 API 设置验证）
  • 使用 "测试连接" 按钮验证引擎可用性
  • 确认网络连接正常
  • 默认 Gemini 需确保环境变量 VITE_API_KEY 已设置
Q3: 如何重置所有设置？ ▼

清除浏览器 localStorage：

打开浏览器开发者工具 → Application → Local Storage
→ 删除 zen_settings、zen_text 等 key
Q4: 如何在绿幕/直播场景使用？ ▼
  1. 将背景透明度设为 0%，背景色设为绿色（HSL: 120°, 100%, 50%）
  2. 调整文字颜色和字号的对比度
  3. 使用 OBS 等软件的色度键（Chroma Key）过滤绿色背景
Q5: 应用支持哪些语言？ ▼
  • 界面语言：中文（简体）
  • 语音识别：中文（普通话），可通过修改 recognition.lang 适配其他语言
  • AI 生成脚本：中文
